Wizard's Chess

I am currently working on a game of wizards chess where the shape of the board can me modified at the start of the game and at runtime. Rules can also be altered by spells to make this a unique chess experience. I have configured this project in a way that supports machine learning for AI as well as more traditional chess engines.
You can try out a prototype version of this project HERE.

Battle Formations (Utility AI)

A previous project inspired by Total War and MOBAs was experimenting with a game where you would control small to medium sized formations of units and have them move around you, the commander. This project used Utility AI and a simple state machine to control the units within the formations and reinforce fallen soldiers(Demonstrated in slow motion in DemoReel).

Project Voxel

A partner and I worked on a project which featured procedural mesh generation and conversion of MagicaVoxel files to OBJ. While my partner worked on the backend, I focused on the gameplay, art, and experimenting with various genres. I would love to work on and with systems like this in the future.

Action RPGs

I have implemented character controllers using Finite State Machines. Features included melee and projectile weapons, shields, swimming, knockbacks, dodges, special attacks, ledge gripping and climbing, object carrying, and a few other minor features. I've spent an extensive amount of time throughout multiple action rpg projects with animations, fluid transitions, and physics.
